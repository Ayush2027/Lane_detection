WHAT IS LANE DETECTION?

Lane detection is the process of detecting white lines on the roads. Lane tracking is the process of assisting the vehicle to remain in the desired path, and it controls the motion model by using previously detected lane markers.

INTRODUCTION

Lane detection is a useful way to detect driving lanes in self driving cars. This has been made using principles of computer vision such as edge detection, special filtering, frequency distribution and appropriate filters. All of this has been done after converting colored video to greyscale. The main reason for converting from RGB to greyscale is that it simplifies the algorithm and reduces computational requirements. Further Hough transform works for line detection using the hough line transform method. It can detect the shape even if it is broken or distorted a little bit. A combination of all the above mentioned techniques gives smooth lane detection that finds it uses in many AI powered vehicles.

Using a value ranging from 0 (black) to 255, the degree of brightness is described (white). Images that are grayscale instead of RGB tend to have less information. However, employing a grayscale image saves space and is quicker, especially when dealing with complex computations, therefore they are frequently used in image processing.

KSIZE

Our target picture for normalisation is specified by the img option (reduce noise). The ksize parameter specifies the size of the kernel that we will use to convolute (pass over) the image. The noise reduction is accomplished using the kernel convolution. The Gaussian Kernel, which is used for normalising images, is the kernel that this function employs.

SIGMA PARAMETER

The sigma parameter specifies the standard deviation along the x-axis. The image's pixel spread is measured by the standard deviation. A 0 standard deviation is desired because we want the pixel spread to be uniform.

DETECTING EDGE

Canny Filter is the main filter used for edge tecetion as it separated the high frequencies from the lower ones. The higher frequencies show high contrast. As we are aware from basics of image and video processing that edges are areas of high contrast. For better accuracy there must be lesser false positives and false negatives. One thing to notice in this project is that gaussian filtering is used in place of box filter. Box filter is often used in object detection as it helps to reduce random noise but in our case rather than random noise we are more concerned about removing low frequencies and getting high frequency values(edges). To filter according to frequencies it is important to use gaussian filtering.

MAIN USES

The main use of lane detection lies in self driving cars. High accuracy as well as satisfaction will be achieved only after removing noise and dealing with all frequencies in the video.
